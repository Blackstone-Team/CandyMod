# CandyMod
## Mod based on candies

Have you ever wanted to be in world of candies? Well, now you can. Jump into the world of cuteness and eternal happiness! 

> No mobs\
> No danger\
> **_`ONLY SWEETNESS`_**

We'll see you on the other side of rainbow!
